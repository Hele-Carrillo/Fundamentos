3. Flexbox y Grid

Flexbox (Flexible Box Layout)

Ideal para alinear elementos horizontal o verticalmente.

Paso 1: Crear un contenedor flexible

.container { display: flex; }

Propiedades útiles:

  • justify-content: alinea horizontalmente (flex-start,center,space-between,space-around)
  • align-items: alinea verticalmente (flex-start,center,stretch)
  • flex-direction: dirección de los elementos (row, column)

CSS Grid

Ideal para diseños más complejos con filas y columnas.

.grid-container { display: grid; grid-template-columns: 1fr 2fr; grid-gap: 20px; }

Propiedades:

  • grid-template-columns: define columnas.
  • grid-template-rows: define filas.
  • grid-area: permite colocar elementos por nombre o coordenadas.
