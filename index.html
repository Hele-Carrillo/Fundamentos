1. Box Model

Es la forma en que CSS estructura todos los elementos como cajas rectangulares. Entender el modelo de caja es clave para controlar el espacio entre elementos y su dise√±o.

Partes del modelo de caja:

      
  • Contenido (content): el texto o imagen dentro del elemento. 
     
    •  
  • Relleno (padding):espacio entre el contenido y el borde.
    •  
  • Borde (border): el marco que rodea al padding.
    •  
  • Margen (margin): espacio entre el borde del elemento y otros elementos.
Ejemplo visual de box model
(width: 200px; padding: 10px; border: 2px solid black; margin: 20px)

Propiedad box-sizing:

content-box (por defecto): ancho y alto solo incluyen el contenido.
border-box: el ancho y alto incluyen padding y border.

* {
  box-sizing: border-box;
}
