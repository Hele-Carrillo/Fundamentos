2. Posicionamiento y Visualización

Display block

Controla cómo se muestra un elemento.

  • block: ocupa todo el ancho. (Ej. div, p )
  • inline: se comporta como texto. (Ej. span )
  • inline-block: como inline, pero acepta alto y ancho.
  • none: oculta el elemento.

Position

Controla la ubicación de un elemento.

  • static: posición normal (por defecto).
  • relative: se puede mover respecto a su posición normal.
  • absolute: se posiciona respecto al contenedor con position: relative.
  • fixed: se queda fijo en la pantalla al hacer scroll.
  • sticky: se comporta como relative hasta cierto punto, luego se queda fijo.
Ejemplo: cabecera fija con position: fixed

z-index

Controla qué elemento se superpone. Cuanto mayor el número, más arriba estará.
